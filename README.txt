== Description ==

Český seznam přání s možností aktivace hlídacího psa zvýší vaše tržby bez nutnosti složitého kódování. Zákazník 2 kliknutími přidá produkt do svého wishlistu. Na rozdíl od jiných seznamů přání zde může ukládat i tipy na dárky pro svoje přátele.

= Tomikup sdílí odkazy na váš e-shop: =

Nová přání Tomikup aktivně rozešle přátelům vašeho zákazníka. Odkazy na produkty z vašeho e-shopu jim připomene i před jeho narozeninami a svátkem.

= Jediná registrace pro všechny e-shopy: =

Zákazníci nejsou obecně ochotní registrovat se kvůli wishlistu ve vícero e-shopech. Tomikup wishlist si však vystačí s jedinou registrací pro všechny e-shopy. 50 tisíc uživatel Tomikupu si snadno přidá na seznam přání vaše produkty bez nutnosti přihlášení či dokonce registrace. Ostatní se zaregistrují jediným kliknutím.

= Hlídací pes - Hlídání cen a dostupnosti: =

Jednoduše a bez programování můžete svým zákazníkům nabídnout hlídání slev. Stačí Tomikupu dodat svůj zbožový feed ve formátu pro Google nákupy (<a href="https://www.tomikup.cz/stat/e-shopy-wishlist">více info o hlídacím psovi</a>). 

Uživatel i jeho přátelé jsou pak upozorňováni na slevy a každý pokles ceny u produktů, které si přidali na wish list nebo seznam dárků.

Tomikup wishlist automaticky mění jazyk dle jazyka šablony. Seznam přání aktuálně mluví česky a, slovensky.

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'Timikup Wishlist'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `tomikup-wishlist.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `tomikup-wishlist.zip`
2. Extract the `tomikup-wishlist` directory to your computer
3. Upload the `tomikup-wishlist` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
